Arrays

Los arrays se representan como [N]T, donde N es el número de elementos en el array y T es el tipo de esos elementos (es decir, el tipo hijo del array).

Para literales de array, N puede ser reemplazado por _ para inferir el tamaño del array.

const a = [4]u8{ 'h', 'o', 'l', 'a',  };
const b = [_]u8{ 'm', 'u', 'n', 'd', 'o' };

Para obtener el tamaño de un array, simplemente accede al campo len del array.

const array = [_]u8{ 'h', 'o', 'l', 'a' };
const length = array.len; // 4
